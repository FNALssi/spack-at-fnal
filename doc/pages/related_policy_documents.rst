Policy Documents Related to Spack
=================================

The documents below describe policies related to the use of Spack in CSAID.
Click on the link to download the PDF document.

* :download:`A Plan for Standard Builds of LArSoft <../static/plan-for-standard-builds.pdf>`
  This file describes the plan for creating standard releases of LArSoft using Spack.
  Other experiments using part of the same software stack may also find this document useful.
  A similar plan will be developed for other software stacks supported by CSAID.

* :download:`A Plan for Maintaining Spack Recipes <../static/recipe_maintenance.pdf>`
  This document describes the CSAID plan for the maintenance of Spack recipes.
